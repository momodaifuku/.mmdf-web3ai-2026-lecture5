A-1 プロダクト Web App

Memoria

アイドルカメコ必見！ 深夜の5000枚の選別地獄を、 直感的なスワイプで10分の「無心ゲー」に変える写真整理アプリ

プロダクトのスクリーンショット（<img src="figures/hero.png">等）を配置

01 バグ 実際に困ったこと 必須

「ライブの日の深夜、SDカードがパンパンになり、PCの前で大量の似たような写真を前に何時間もため息をついていた」

  • 撮影直後で疲労困憊のアイドルカメコ
  • 既存ソフトは多機能すぎて操作手順が多く、判断疲れを起こすから

02 解決策 (プロダクト) 選んだ形 必須

直感的なスワイプ操作によって、疲れた深夜でも無心で数千枚の写真をサクサク選別し、最速で楽しいレタッチ作業へと移れるようになる

スワイプ画面のスクリーンショット等を配置

03 検証実績 試して確かめた 必須

3名 使ってくれたカメコ仲間
5回 自身で試した回数
5000枚 整理できた写真の数
10分 完了までにかかった時間
Before 数時間（ため息）
→
After 10分（無心でサクサク）
「レタッチする写真の選定がめっちゃ楽になった！」 — カメコ仲間

04 技術構成 賢さは借りて、力仕事は自分で 推奨

数千枚の重い写真をサーバーにアップロードせず、ブラウザ上で処理完結するローカルJS処理（File API / URL.createObjectURL）を採用。

  • Vanilla JS / CSS高速なローカル処理・グラスモーフィズムUI
  • Hammer.jsネイティブアプリのような直感スワイプ
  • AI (LLM)アイデアを爆速でコードへと翻訳するペアプログラマ

05 失敗と学び 失敗は学びのたね 必須

✗ 通用しなかった

「ゲーム感覚」を狙った制限時間制度（30秒固定）は焦ってしまい微妙だと不評だった

✓ 分かったこと

ユーザーの用途に合わせて自由に時間を選択できるようにすると自分のペースで進められると分かった

06 次の一手 (方向転換案) もっとよくしていく 強く推奨

連写による似た画角の写真の中から、半目や口開きなどをAIで自動判定して弾く機能を実装したい。
